## Project Overview

LuxeCart is a premium e-commerce web application developed for the Thiranex Web Development Internship Task 5. The project demonstrates front-end product rendering, cart state management, checkout validation, local storage persistence, responsive UI design, and a dashboard-style business overview.

This project is designed to feel like a real digital commerce product rather than a basic assignment.

## Key Features

- Premium e-commerce landing page
- Product catalog with search and category filtering
- Add to cart functionality
- Quantity increase and decrease controls
- Remove cart items
- Subtotal, GST, and total calculation
- Checkout form validation
- Mock order placement flow
- Admin-style dashboard cards
- Light/Dark theme toggle
- Local Storage state persistence
- Fully responsive layout for mobile, tablet, and desktop

## Technologies Used

- HTML5

- JavaScript
- CSS Grid

- Local Storage API

- Responsive Media Queries
